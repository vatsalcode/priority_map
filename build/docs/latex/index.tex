The {\ttfamily priority\+\_\+map} class is designed to manage a set of elements, each with a priority, ~\newline
 allowing for efficient retrieval, updating, and tracking of elements based on their priority. ~\newline


It is designed to provide constant time access to the element with the highest priority, ~\newline
 as well as constant time priority increment and decrement. ~\newline


Setting priority for a given element directly is O(n). ~\newline


The underlying implementation is a doubly linked list backed by a hashmap. ~\newline
